## HTML Sêmantico
``` html

Header = Inicio da Página
```

## Controladores de Texto
``` html

U = Sublinhado;
Hr = Linha;
Br = Break;
Strong = Negrito;
P = Parágrafo;
I = Itálico;
H1... H6 = Títulos;

```

## Controladores de Conteúdo

``` html

Div = Divisão da Página;
IMG scr = Imagem;
Width = Largura;
Height = Altura;
UL = Lista Desordenada;
OL = Lista Ordenada;
LI = Nova linha na lista;

```
